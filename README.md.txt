# Bantaba Marketplace

The best online marketing platform for The Gambia 🇬🇲 - Buy & Sell with WhatsApp integration.

## Features

✅ **Easy Login** - Phone number + name, no complicated authentication  
✅ **Quick Selling** - Post products in seconds (5 simple fields)  
✅ **Low Data** - Optimized for slow internet connections  
✅ **WhatsApp Integrated** - Share products & contact sellers directly  
✅ **Favorites System** - Save products you like  
✅ **Ratings & Reviews** - Trust-based marketplace  
✅ **6 Categories** - Electronics, Vehicles, Food & Agriculture, Fashion, Self Care, Property  

## Getting Started

### Prerequisites
- Flutter SDK 3.0+
- Dart

### Installation

```bash
# Clone the repository
git clone https://github.com/kambi3427911/bantaba-marketplace.git
cd bantaba-marketplace

# Install dependencies
flutter pub get

# Run the app
flutter run
